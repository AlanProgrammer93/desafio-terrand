# 🌟 Desafío- Desarrollador Fullstack.

## Crea una aplicación donde los usuarios puedan registrarse y compartir recetas de cocina.
- Registro
    Nombre, Apellido, Email, Contraseña y Repetir contraseña
- Login
    Email y Contraseña
- Sección privada para ver todas mis recetas, editar y crear una nueva
- Un receta debe tener como mínimo:
    Título, Descripción, Ingredientes
- La aplicación debe generar un link público donde se pueda ver la receta

## 📌Opcionales:
- Permitir que los usuarios logueados califiquen las recetas.
- Que cada receta permita subir una imagen
- Publica tu aplicación (Deploy)

## 👉Requerimientos técnicos:
- Utilizar cualquier tecnología de JavaScript

## 🔍 Qué evaluaremos
- Lógica
- Manejo de Estado
- Componentización
- Modelado de la base de datos
- Legibilidad y calidad del código
